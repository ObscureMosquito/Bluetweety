<h1>BlueTweety</h1>
This is the BlueTweety Tweak, as hosted in bag.xml and Requi's repo, it redirects URLs from the App Store Twitter App and iOS Twitter native integration to to any domain you specify in settings (check main branch for hosting).
</div>

&nbsp;

## Usage & Tutorial

#### Download and install the tweak from my Cydia repo, bag.xml's Cydia repo:

```bash
https://cydia.skyglow.es
```

```bash
https://cydia.bag-xml.com
```

### To run Bluetweety, make sure:

1. You have entered a valid Bluesky-Bridge URL in the preference pane and restarted the app.
1. You have the latest Bluetweety version.
1. You have a relative stable internet connection.

#### Also check out the [server itself](https://github.com/Preloading/TwitterAPIBridge).
